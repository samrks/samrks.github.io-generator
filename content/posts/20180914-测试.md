## 内置 Shortcodes 语法支持
### 横幅

​	```markdown
{{< admonition type=danger title="This is a tip" >}}
一个 danger 横幅
{{< /admonition >}}

{{< admonition type=note title="This is a tip" open=false >}}
一个 note 横幅
{{< /admonition >}}

{{< admonition tip "This is a tip" true>}}
一个 tip 横幅
{{< /admonition >}}
```

{{< admonition type=danger title="This is a tip" >}}

{{< /admonition >}}
